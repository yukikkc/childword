# 子育て英単語学習アプリ

子育てに関連する英単語を音声付きで学習できるWebアプリケーションです。

## 特徴

- 📚 **4,000語の豊富な語彙**
  - 医療・健康、日常動作、食事、遊びなど子育てに必要な単語を網羅
  - 日本語と英語の例文付き

- 🔊 **音声読み上げ機能**
  - Web Speech APIによる自動音声読み上げ
  - 読み上げ速度調整可能
  - カスタマイズ可能な読み上げパターン（英→日、日→英、繰り返し回数など）

- ⚙️ **多彩な学習機能**
  - カテゴリー別学習
  - ランダム再生モード
  - ブックマーク機能（復習したい単語を保存）
  - 単語検索機能
  - 自動再生モード（スリープ防止機能付き）

- 📱 **スマホ対応**
  - レスポンシブデザイン
  - スワイプ操作対応
  - PWA対応（ホーム画面に追加可能）

## 使い方

### オンラインで使用

1. `index.html`をブラウザで開く（スマホ版）
2. または`pc.html`をブラウザで開く（PC版）

### オフライン（Wi-Fi/ネット接続なし）で使用

このアプリは**完全オフライン対応**です。以下の3つの方法で使用できます。

**📱 iPhoneでの使い方は [iPhone使用ガイド](IPHONE_GUIDE.md) をご覧ください。**

#### 方法1: HTMLファイルを直接開く（最も簡単・完全オフライン）

**スマホの場合:**
1. GitHubから`index.html`と`vocabulary.js`をダウンロード
2. スマホのファイルアプリで保存
3. `index.html`をタップしてブラウザで開く
4. **Wi-Fi OFF、機内モードでも動作します** ✈️

**PCの場合:**
1. GitHubから全ファイルをZIPでダウンロード
2. 展開して`index.html`をダブルクリック
3. **インターネット接続不要で動作します**

#### 方法2: スマホのホーム画面に追加（PWA）

**iPhone/iPad（Safari）:**
1. 一度オンラインでアプリを開く
2. 共有ボタン（□に↑）→「ホーム画面に追加」
3. 以降、Wi-Fi OFFでも起動可能

**Android（Chrome）:**
1. 一度オンラインでアプリを開く
2. メニュー（⋮）→「ホーム画面に追加」
3. 以降、Wi-Fi OFFでも起動可能

#### 方法3: ローカルサーバーで使用（開発者向け）

```bash
# リポジトリをクローン
git clone https://github.com/yukikkc/childword.git
cd childword

# Pythonでローカルサーバーを起動
python3 -m http.server 8000

# ブラウザで http://localhost:8000 を開く
# Service Workerがキャッシュ後、サーバー停止してもオフライン使用可能
```

**📌 推奨:** 完全オフライン環境では **方法1（HTMLファイルを直接開く）** が最も確実です。

### 主な操作

- **スワイプ**: 左右にスワイプで単語切り替え
- **▶ 再生**: 現在の単語を読み上げ
- **▶ 自動**: 自動再生モード（全単語を連続再生）
- **⚙️ 設定**: カテゴリー選択、読み上げ設定など
- **☆ ブックマーク**: 復習したい単語を保存

## 注意事項

### 音声読み上げについて
- 音声読み上げはブラウザの**Web Speech API**を使用しています
- **オフライン使用時の音声について：**
  - **Safari（iOS/Mac）**: オフラインでも音声読み上げが可能
  - **Chrome/Edge**: 一部の音声はオンライン接続が必要な場合があります
  - 完全にオフラインで使用する場合は、事前にオンラインで一度起動し、音声データをキャッシュすることをお勧めします

### その他
- 自動再生モード使用時はバッテリー消費が増加します
- 画面スリープ防止機能により、バッテリー消費が通常より多くなります

## 開発者向け情報

### ファイル構成

```
childword/
├── index.html          # メインアプリ（スマホ版）
├── pc.html            # PC版
├── vocabulary.js      # 4,000語の単語データ
├── manifest.json      # PWA設定ファイル
├── sw.js              # Service Worker（オフライン対応）
├── IPHONE_GUIDE.md    # iPhone使用ガイド
├── .gitignore         # Git除外設定
├── LICENSE            # MITライセンス
└── README.md          # このファイル
```

**📦 最小構成（完全オフライン使用）:**
```
childword/
├── index.html      # 必須
└── vocabulary.js   # 必須（4000語）
```
この2ファイルだけでWi-Fiなしでも動作します。

### カスタマイズ

`vocabulary.js`の単語データは自由に編集できます。形式：

```javascript
{
    category: "カテゴリー名",
    jp: "日本語",
    en: "English",
    exampleJp: "日本語の例文",
    exampleEn: "English example sentence"
}
```
