# Wi-Fiなしで使う方法（完全オフライン）

このアプリは**Wi-Fi・モバイルデータなし**の完全オフライン環境で使用できます。

## 📱 スマホで使う（最も簡単）

### ステップ1: ファイルをダウンロード

1. GitHubのリポジトリページを開く
2. 緑色の「Code」ボタン → 「Download ZIP」
3. ZIPファイルをダウンロード
4. ダウンロードしたZIPを展開（解凍）

### ステップ2: ファイルをスマホに保存

**iPhone/iPad:**
- 「ファイル」アプリで「ダウンロード」フォルダを開く
- ZIPファイルをタップして展開
- `index.html` と `vocabulary.js` が同じフォルダにあることを確認

**Android:**
- ファイルマネージャーアプリを開く
- ダウンロードフォルダを開く
- ZIPファイルを展開
- `index.html` と `vocabulary.js` が同じフォルダにあることを確認

### ステップ3: アプリを開く

1. `index.html` をタップ
2. ブラウザで開くアプリを選択（Safari または Chrome推奨）
3. **Wi-Fi OFF・機内モードでも動作します！** ✈️

## 💻 PCで使う

1. GitHubから全ファイルをZIPでダウンロード
2. 任意のフォルダに解凍
3. `index.html` をダブルクリック
4. ブラウザで開く
5. **インターネット接続不要！**

## ✅ 動作確認

完全オフラインで動くか確認：

1. スマホを**機内モード**にする ✈️
2. Wi-Fiを**OFF**にする
3. `index.html` を開く
4. 音声読み上げが動作するか確認

**音声について:**
- Safari（iPhone）: オフラインでも音声が出ます ✅
- Chrome/Edge: 初回はオンライン接続が必要な場合があります

## 📝 必要なファイル

最小限、以下の2ファイルがあれば動作します：

```
child-word/
├── index.html      ← 必須
└── vocabulary.js   ← 必須（4000語のデータ）
```

その他のファイル（`sw.js`, `manifest.json` など）はオンライン機能用なので、
完全オフライン環境では不要です。

## 🎯 よくある質問

**Q: Wi-Fiがないと音声は出ませんか？**
A: Safari（iPhone/iPad）なら完全オフラインでも音声が出ます。
   Chrome系ブラウザは一部制限がある場合があります。

**Q: ダウンロードは何度も必要ですか？**
A: 一度ダウンロードすればずっと使えます。
   単語データの更新がある場合のみ再ダウンロードしてください。

**Q: スマホの容量はどれくらい必要ですか？**
A: 約1MB程度です（vocabulary.jsが約800KB）。

**Q: インストールは必要ですか？**
A: 不要です。HTMLファイルを開くだけで使えます。

## 🔄 更新する方法

新しい単語が追加された場合：

1. GitHubから最新の `vocabulary.js` をダウンロード
2. 古いファイルと置き換える
3. 完了！

`index.html` は基本的に更新不要です。

---

**💡 ヒント:** ブックマーク機能を使えば、復習したい単語を保存できます。
データはブラウザに保存されるため、オフラインでも使えます。
